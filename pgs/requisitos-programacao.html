Quais são os pré-requisitos para começar a programar

- Requerimentos físicos (hardware)
Usando um notebook para programar
Para poder programar, primeiramente é necessário um computador com sistema operacional Windows ou MacOS, ou, até mesmo um celular smartphone. Na verdade, você pode escrever código com papel e lápis, só não é útil para poder testar, por isso nós recomendamos o uso de hardware; computadores de qualquer tipo, seja um desktop, notebook ou smartphone. Um artista sem pinceis e tinta é como um programador sem um computador!
- Requerimento de programas (software)
Assim que tiver seu hardware, é preciso instalar algum tipo de programa compatível com ele para podermos desenvolver, afinal, recomendamos começar a aprender com uma linguagem de “alto nível”, isso significa que escreveremos código num IDE (Integrated Development Environment - Ambiente de Desenvolvimento Integrado), que irá então traduzir (compilar) nosso código para um formato que o computador entenda, os famosos “0 e 1s” que chamamos de binário.
Um desenvolvedor pode escolher uma (ou mais!) linguagem de programação de sua preferência para poder escrever o código de um programa, ele pode tomar essa decisão baseado nas tecnologias e possibilidades de cada língua, além do seu nível de conhecimento em cada uma.
Mas o que é uma linguagem de programação?
Trecho de programa em linguagem C
Uma linguagem de programação, é como uma lista de instruções para uma máquina. O computador, ao receber essa lista de instruções, executa cada comando como foi escrito. Infelizmente para os desenvolvedores, programar não é tão simples quanto escrever num computador:” Crie uma calculadora científica”. Assim como no texto de uma língua escrita, existem regras a serem seguidas, que chamamos de sintaxe e a semântica, talvez você seja familiar com essas palavras, e o seu paralelo no mundo da gramática é muito similar.
Alto e baixo nível
Linguagens de alto nível são preferidas por desenvolvedores por ser mais fácil de ler e compreender, o que facilita o desenvolvimento de programas mais sofisticados. Enquanto isso, linguagens de baixo nível focam em manipular dados na sua forma mais próxima do processador, e por isso se limitam em portabilidade, um código de baixo nível que funciona em um computador não irá funcionar em outro que tem um processador com arquitetura diferente.
É importante saber o vocabulário dessa área do conhecimento, e podemos no futuro criar um artigo detalhando mais profundo, mas conhecer o vocabulário e importante são só para a sua própria organização, mas também para trabalhar em grupo, e explicar uma dúvida.
Onde Tirar dúvidas
Logo do stackoverflow
Programar!
A grande maioria dos programadores ajudam uns aos outros pelo site Stack Overflow (stackoverflow.com), cujas principais funções são pesquisar e responder dúvidas de outros usuários, além de poder postar suas próprias. A comunidade é grande num nível global, dos iniciantes aos experientes do mercado. O único requisito para usar bem o site é ter um certo nível de fluência em Inglês
